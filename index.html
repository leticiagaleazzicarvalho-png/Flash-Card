Ver resposta

HTML

Não! O HTML é uma linguagem de marcação usada para estruturar e dar corpo aos elementos da página.

Aula 3

Por que a hierarquia de tags (H1 a H6) é importante?

Ver resposta

Hierarquia

Ela organiza a importância dos títulos para o leitor e ajuda os motores de busca (como o Google) a entender o site.

Aula 4

Qual é o papel principal do CSS no desenvolvimento?

Ver resposta

Estilização

O CSS cuida da estética: define cores, fontes, tamanhos e toda a parte visual do esqueleto criado no HTML.

Aula 5

Para que serve o modelo Flexbox no CSS?

Ver resposta

Layout Flexível

Serve para alinhar e distribuir elementos em um container de forma simples, adaptável e responsiva.

Aula 6

Como podemos representar cores no CSS além do nome em inglês?

Ver resposta

Cores e Estilos

Podemos usar códigos Hexadecimais (ex: #ffffff) ou o sistema RGB (ex: rgb(255, 255, 255)).

Aula 7

O que faz a propriedade position: absolute?

Ver resposta

Posicionamento

Remove o elemento do fluxo normal e o posiciona de forma exata em relação ao seu elemento pai mais próximo.
